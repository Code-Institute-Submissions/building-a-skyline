Welcome to Cities: Skylines!

Can I run the game?

So you are thinking of buying the game or you just started and you need some guidance where to go next? Before you go on, have in mind that Cities: Skylines is a game that came out in 2015 and since then it had a lot of DLC's (downloadable content) that expands on base game. So if you want to start, start with the base game, and then add DLC's you like the most.

Minimum requirements:
  • Requires a 64-bit processor and operating system
  • OS: Microsoft Windows XP/Vista/7/8/8.1 (64-bit)
  • Processor: Intel Core 2 Duo, 3.0GHz or AMD Athlon 64 X2 6400+, 3.2GHz
  • Memory: 4 GB RAM
  • Graphics: nVIDIA GeForce GTX 260, 512 MB or ATI Radeon HD 5670, 512 MB (Does not support Intel Integrated Graphics Cards)
  • Storage: 4 GB available space
Other platforms:

You can also play Cities: Skylines on following platforms:

  • Playstation 4
  • Xbox One
  • Nintendo Switch

Have in mind that DLC availability is not the same for consoles and PC/Mac
